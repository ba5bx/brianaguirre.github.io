Frnnfluence Javascript, Grunt, SASS, Node, Twilio API

Frnnfluence (pronounced as if 'friend' and 'influence' were mashed together) is an app that I am currently working on with @helenvholmes. A simple SMS app, Frnnfluence allows you to announce when and where you're working out so that friends can RSVP. Unlike announcing it on twitter or facebook, our app was meant to be small, light-weight (since it's an SMS app, a larger audience can use it) and meant only to text close friends that you actually want to know when and where you're working out.

While the app is by no means finished, because what was built was done over the course of a day at a hackathon it was an opportunity to try out some new, cool technologies that I haven't had the opportunity to use in school.
